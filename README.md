# Pocket Showroom Core

A modern B2B product catalog WordPress plugin with CSV import, multi-image gallery, and interactive frontend.

## Features

- **Interactive Product Gallery** - Modern grid layout with category filtering and search
- **CSV Bulk Import/Export** - Import hundreds of products instantly
- **Multi-Image Gallery** - Drag-and-drop image upload with sorting
- **Social Sharing** - Beautiful Open Graph cards for WhatsApp/WeChat sharing
- **Image Watermarking** - Automatic watermark on uploaded images
- **AJAX-powered Details** - Product details in modal without page refresh
- **Responsive Design** - Perfect on mobile, tablet, and desktop
- **Auto-Update Support** - Automatic updates from GitHub Releases

## Installation

1. Download the latest release ZIP file
2. Go to WordPress Admin → Plugins → Add New → Upload Plugin
3. Upload the ZIP file and activate
4. Configure settings under **Pocket Showroom** → **Settings**
5. Add shortcode `[pocket_showroom]` to any page

## Requirements

- WordPress 5.0+
- PHP 7.4+

## Usage

### Adding Products
1. Go to **Pocket Showroom** → **Add New**
2. Fill in product details (model, material, size, MOQ, etc.)
3. Upload images to the gallery
4. Publish

### CSV Import
1. Go to **Pocket Showroom** → **Import/Export**
2. Download the CSV template
3. Fill with your product data
4. Upload and import

### Display on Frontend
Add the shortcode to any page:
```
[pocket_showroom]
```

## Auto-Updates

This plugin supports automatic updates from GitHub Releases. When a new version is released:
1. You'll see an update notification in WordPress Admin → Plugins
2. Click "Update Now" to automatically update
3. Or manually check for updates using the "Check for Updates" link
